# Skill: Assemble Draft System Trail

Use only after the learner has selected, edited, or created candidate system parts.

## Task

Use only the selected candidate parts to create an ordered draft System Trail.

## Rules

- Do not add major new parts unless needed for coherence.
- If a part is added, mark it as `ai-suggested`.
- Preserve learner-created or learner-edited parts.
- Keep each node as visible behavior.
- Put the simplest visible first milestone first.
- Order the remaining parts toward the end goal.
- Add simple sequence, dependency, or feedback-loop edges.
- This is still a draft and must be reviewed.

## Output

Return `DraftSystemTrailResult`.
