# Skill: Suggest First Milestone

Use after the learner has reviewed the draft System Trail.

## Task

Recommend 1-3 good first milestones from the reviewed draft trail.

A good first milestone should:

- be easy to start;
- create visible progress;
- connect clearly to the learner's goal;
- be small enough to build and preview;
- not require too many other parts first.

Do not choose automatically. The learner must pick.

## Output

Return `FirstMilestoneRecommendationResult`.
