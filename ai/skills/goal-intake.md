# Skill: Goal Intake

Use when the learner has entered an open-ended idea and the planning session has just started.

## Task

- Restate the learner's idea in simple language.
- Create a short project title.
- Identify the broad artifact type only if obvious.
- Do not generate a full System Trail.
- Do not generate code.
- Prepare the learner to start co-planning.

## Output

Return `GoalIntakeResult`.

## Quiet AI Style

One short supportive sentence. Make it clear that the system will be made visible before building.
